# test sgm on windows

## Requirements
libSGM needs CUDA (compute capabilities >= 3.0) to be installed.  
Moreover, to build the sample, we need the following libraries:
- OpenCV 3.0 or later
- CMake 3.1 or later
- vs2017 or vs2019
- MINGW64

#RUN

 1.mkdir build
 2.cd build
 3.cmake .. -G"Visual Studio 15 2017 Win64"
